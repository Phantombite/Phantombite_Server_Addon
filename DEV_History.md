# DEV History — PhantomBite Server Addon

## 2026-03-22 — v1.0.0 — Initialer Release

- Aus NimbusMod extrahiert und als eigenständiger Mod veröffentlicht
- Steam Workshop ID: 3689667750
- GitHub Repository: https://github.com/Phantombite/PhantomBiteServerAddon

### Hintergrund
Der NimbusMod war ursprünglich ein Sammelmod für alle Server-Anpassungen. Durch die Trennung sind die Vanilla-Anpassungen, das Radiation Kit und der Fraktionsschutz nun in einem eigenen schlanken Mod gebündelt der unabhängig von anderen PhantomBite Mods genutzt werden kann.
